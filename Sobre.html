Partes do Projeto

Início do Projeto:

  • No início do projeto, foi utilizado principalmente HTML e CSS para estabelecer toda a base do site

Desenvolvimento:

  • Durante o desenvolvimento, foram implementadas diversas animações utilizando JavaScript, adicionando dinamismo e interatividade ao site. Além disso, avancei na integração PHP com MySQL para salvar informações importantes, como nome e detalhes do veículo.

Finalização:

  • Na fase final do projeto, o foco foi refinado para garantir a funcionalidade robusta e a usabilidade do site, superando desafios iniciais e maximizando o potencial das tecnologias empregadas.
